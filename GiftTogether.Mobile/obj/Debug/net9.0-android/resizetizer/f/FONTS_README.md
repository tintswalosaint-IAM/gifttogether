# Inter Font Files

Download Inter font files from https://fonts.google.com/specimen/Inter and place them here:
- Inter-Regular.ttf (weight 400)
- Inter-Medium.ttf (weight 500)
- Inter-SemiBold.ttf (weight 600)
- Inter-Bold.ttf (weight 700)

These are used as offline fallbacks. The app loads Inter from Google Fonts CDN at runtime.
